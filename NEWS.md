## rdynamicscrm 0.0.0.9000

### Features

  * Auto-refreshing authentication method (`dyn_auth()`)
  * Query operation (`dyn_query()`)
  * CRUD operations (Create, Retrieve, Update, Delete):
    * `dyn_create()`
    * `dyn_retrieve()`
    * `dyn_update()` 
    * `dyn_delete()`
  * Basic utility calls: 
    * `dyn_whoami()`
